# JFrog Learn — Artifactory & Xray, Bottom-Up

An interactive, self-paced learning site that teaches **JFrog Artifactory** and **JFrog Xray** from first principles — no prior knowledge assumed.

## What's inside
- **Foundations** — the problem these tools solve + a vocabulary primer (artifact, CVE, SCA, SBOM, JPD, build-info, promotion).
- **Artifactory** — what it is, the 4 repository types (local/remote/virtual/federated) with diagrams & tables, pipeline placement, and supported package types.
- **Xray** — deep recursive scanning, the scan lifecycle, and the Policy vs. Watch model.
- **Hands-on** — three lab tracks: JFrog Cloud free tier, local Docker self-hosted, and Concepts + CLI, plus a `jf` CLI cheat sheet.
- **Synthesis** — a one-screen mental model and a 4-point recap.

## Run locally
It's a static site — just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Files
- `index.html` — content & structure
- `style.css` — design system (light/dark, responsive)
- `app.js` — theme toggle, mobile nav, tabs, scrollspy

All facts link to primary JFrog documentation. Product names belong to JFrog Ltd.
